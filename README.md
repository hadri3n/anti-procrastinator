anti-procrastinator
===================


###Run the application with
`gradle bootRun --daemon` for 'dev' profile

or

`gradle bootRun -Pprofile=my_profile --daemon`


### Run tests with Spring MockMVC
`gradle mockMvcTest`

### Run tests with Spock
`gradle spockTest`

### Run tests with Cucumber
`gradle cucumberTest`

### Run all tests
`gradle check`

### Run all tests
`gradle check`

### Run tests code coverage
`gradle cobertura`
